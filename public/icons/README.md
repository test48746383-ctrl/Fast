To ensure the PWA works correctly on all devices (iOS, Android, Desktop), you must place the icon files in this folder: `public/icons/`

## Required Files

Please generate or design square icons in PNG format with the following filenames:

1. `icon-72x72.png`
2. `icon-96x96.png`
3. `icon-128x128.png`
4. `icon-144x144.png`
5. `icon-152x152.png` (Critical for iOS)
6. `icon-192x192.png` (Standard Android)
7. `icon-384x384.png`
8. `icon-512x512.png` (Play Store / Splash)
9. `maskable-icon-512x512.png` (Adaptive Icon)

## Tips
- Use a tool like [Favicon Generator](https://realfavicongenerator.net/) or [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator).
- Ensure the background is `#1a1a2e` or transparent (if maskable).
- For `maskable-icon`, ensure the logo is within the "safe zone" (center 60%).
